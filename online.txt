Đây là dòng đầu tiên được viết trực tiếp trên GitHub web editor.
Tính năng web editor giúp chỉnh sửa code mà không cần clone về local.
GitHub web interface rất tiện lợi cho việc sửa nhanh documentation và config files.
Bây giờ tôi đang chỉnh sửa file này từ local environment sau khi pull về.
Workflow này thể hiện sự đồng bộ hoàn hảo giữa local Git và remote GitHub repository.

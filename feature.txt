# Tính năng: Hệ thống xác thực người dùng thông minh

## Mô tả tính năng
Phát triển hệ thống đăng nhập và xác thực người dùng với tích hợp đa phương thức bảo mật.
Tính năng hỗ trợ đăng nhập bằng email/password, xác thực 2 bước (2FA) qua SMS và Google Authenticator.
Hệ thống sẽ tự động phát hiện và chặn các lần đăng nhập bất thường từ địa chỉ IP khác lạ.
Người dùng có thể quản lý phiên đăng nhập từ nhiều thiết bị và đăng xuất từ xa khi cần thiết.
Tính năng cũng bao gồm khôi phục mật khẩu qua email với mã xác thực có thời hạn 15 phút.

## Chi tiết kỹ thuật
- Backend: Node.js + Express với JWT token authentication
- Database: MySQL để lưu trữ thông tin người dùng và session
- Frontend: React.js với form validation và UI responsive
- Bảo mật: Bcrypt hash password, rate limiting, HTTPS bắt buộc
- Tích hợp: Google Authenticator SDK, SMS gateway API
